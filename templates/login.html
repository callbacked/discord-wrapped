How it Works

1. Add the Bot

Start by adding the Discord Wrapped to your server.

2. Member Opt-in

Members use the /optin command to participate. Only opted-in members will have their activity tracked.

3. Data Collection

In order to produce individual wrapped reports, the bot tracks message activity, reactions, voice time, and more from opted-in members.

4. Personalize

In the dashboard, admins can customize personality traits, rename default ones, and customize card designs.

5. Generate Wrapped

Your Discord Wrapped is sent via DM to opted-in members. Admins can release it anytime using /wrap or the dashboard.
